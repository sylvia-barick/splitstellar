# SplitStellar User Guide

## Getting Started
1. **Install Freighter Wallet**: Install the Freighter extension from [freighter.app](https://www.freighter.app/) and switch network to **Stellar Testnet**.
2. **Connect Wallet**: Click "Connect Wallet" on the top right navigation bar.
3. **Create or Join a Group**:
   - To create a group, navigate to **Groups** -> **Create Group**, fill in group name and default currency (XLM), and share the generated Base32 invite code.
   - To join a group, click **Join Group** and enter the 6-character Base32 invite code.
4. **Add Expenses**: Go to **Expenses** -> **Add Expense**, choose amount and participants.
5. **Inspect Net Debt Graph**: Open **Debt Graph** to view consolidated debts.
6. **Settle Debt On-Chain**: Click **Settle Debt**, confirm the transaction in Freighter wallet, and view your on-chain transaction hash on Stellar Expert Explorer.
